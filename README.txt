# AffineTransformations
5 most used Affine Transformations in Image Processing

This program contains 5 most used Affine transformations in python language which are:
- Scaling
- Rotation
- Translation
- Horizantal Shearing
- Vertical Shearing

Libraries used in this program:
- matplotlib
- numpy
- opencv

Written in:
- Visual Studio 2019
